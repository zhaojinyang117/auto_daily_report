1、### 模板分离技术

把**共用**的html部分给抽出去

比如想在每个页面的右下角就加一条广告

`{% include “ads.html” %}`

ads的html并不需要是一个完整的html，只需要是部分组件即可

但是用很多include会导致模板碎片化的问题

**模板继承**

`{% extends “base.html” %}`

用`{% block name %} {% blockend %}` 和extends共同使用

这样就把样式模板给继承过来了

然后再改`{% block name %} {% blockend %}`中的内容，便可以重写

默认是覆盖base中的block标签

但是可以用`{{ block.super }}`取出父盒的数据

2、其实就是不想把路由写死

比如在urls文件中，给路由加一个`name=ord`参数

**在模板函数的应用**

然后在模板中用 `{% url ord %}` 代替原本应该写死的超链接的位置

**在视图函数的应用**

替代原本写死的重定向

```python
from django.urls import reverse
def f(request):
	path:str = reverse("ord")
	return redirect(path)

```
